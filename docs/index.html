Pramana

Open-source LLM drift detection

The Problem

LLM behavior changes silently. Provider updates, quantization changes, and infrastructure migrations alter model outputs without notice. Chen et al. (2023) documented this empirically: GPT-4's accuracy on identifying prime numbers dropped from 97.6% to 2.4% between March and June 2023. How Is ChatGPT's Behavior Changing over Time? (Stanford/Berkeley) demonstrated that drift is real, measurable, and consequential.

If your application depends on LLM outputs — classification, extraction, generation — you need continuous monitoring, not one-time benchmarks.

What Pramana Does

Pramana is a lightweight eval submission and drift detection system. It consists of two components:

  • CLI (pramana) — runs eval suites locally, scores outputs, submits results
  • API + Dashboard (this repo) — stores submissions, aggregates statistics, detects drift

Detection uses Welch's t-test with Cohen's d effect size and Holm-Bonferroni correction for multiple comparisons. All statistics are computed incrementally via Welford's online algorithm — no raw data is retained.

How It Works

# Install the CLI
pip install pramana

# Run evals against a model
pramana run --model gpt-4o --suite my_evals/

# Submit results
pramana submit

# View drift dashboard
open https://pramana.pages.dev

Data Flow

Each submission follows this path:

  1. POST /api/submit — CLI sends scored results
  2. Append to gzip-compressed CSV buffer in R2
  3. Update per-user summary (real-time)
  4. Daily cron compacts buffer → archive, rebuilds aggregate chart data
  5. Dashboard reads pre-aggregated JSON (single R2 GET, <50ms)

Design Decisions

  • Stateless API — no database, no connection pools. CSV + JSON on R2.
  • Append-only — immutable results. No edits, no deletes of submissions.
  • Incremental statistics — Welford's algorithm: O(1) storage per group.
  • Zero cost — Cloudflare Pages free tier + R2 free tier = $0/month.
  • Privacy-preserving — individual submissions cannot be reconstructed from aggregates.
